# license-to-plate

Simple voting webpage that pairs images against eachother and keeps track of the winners

## Yearly Plate Setup

### Step 1

Download plate zone server images via "discrub" browser extension

### Step 2

Resize all images to same aspect ratio

### Step 3

Rename all image files to "platex.jpg" where x is the plate ID
